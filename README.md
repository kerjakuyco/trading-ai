The core idea — trade like smart money, not like indicators
Instead of reacting to lagging crossovers, this reads market structure the way institutional traders do. The entry model is: liquidity gets swept → structure confirms direction → you enter on the reversal.

How it works, step by step:
- Structure mapping (BOS / CHoCH) — The indicator identifies swing highs and swing lows using pivots, then tracks when price breaks them. A Break of Structure (BOS) means trend continuation. A Change of Character (CHoCH) means a potential reversal — this is the higher-probability setup.
- Liquidity sweep detection — This is the core edge. Before big moves, price often hunts stop-losses sitting below swing lows (for longs) or above swing highs (for shorts). The indicator detects when price wicks past a level but closes back inside — that's smart money grabbing liquidity before reversing.
- Fair Value Gaps (FVG) — Imbalances where price moved so fast it left a gap. These act as magnets — price tends to return and fill them. The indicator draws these zones so you can see where price is likely heading.
- Confluence scoring (4 points):

Structure bias agrees (BOS direction)
Liquidity sweep confirmed
HTF trend alignment (Daily EMA 50 by default)
FVG present nearby

An entry only fires when a sweep occurs AND at least 3/4 confluence points align.

Why this is better for 1H-4H:
On 1H-4H these patterns are much cleaner and respected by real volume.
Recommended settings: Use it on 1H with the Daily HTF filter, or 4H with Weekly. Pivot lookback of 5 works well — increase to 8-10 if you want fewer but stronger levels.
